# Security Policy

## Supported Versions

Security updates for earlier versions are only available via community contributions.

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | :x: |
| 0.2.x   | :x:                |
| 0.1.x   | :x: |
| < 0.0   | :x:                |

## Reporting a Vulnerability

Please report vulnerabilities in the issues tab.
